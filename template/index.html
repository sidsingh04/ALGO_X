Overview

This Project helps the user to visualise the running of basic CPU scheduling algorithms.The 7 Basic CPU Scheduling algorithms are FCFS,SJF,SRTF,Round_Robin,HRRN,Non-Premptive-Priority,Premptive-Priority.


FCFS(FIRST COME FIRST SERVE)

This scheduling algorithm is a non-premtive CPU Scheduling Algorithm which executes task according to the arrival time of the process.Its very simple approach but convey effect and process starvation is high.

SJF(Shortest Job First)

This algorithm executes the process on the basis of their burst time.This is a non-premptive algorithm. Convey effect is minimal and the chances of process starvation is low. This algorithm is difficult to implement in real life as the CPU cannot correctly predict the Burst Time of a process

SRTF(Shortest Remaining Time First)

This is a premptive algorithm which executes the process on the basis of remaining time.Convey effect is very low and there is very less process starvation. This is very efficient CPU scheduling algorithm

ROUND-ROBIN

This is also premptive algorithm, which executes the process for a predefined time quantum in a cyclic process, so that every process gets the chance to execute. There is no convey effect and there is no process starvation.

HRRN(Highest Response Ratio Next)

This is a non-premptive algorithm which executes process on the basis of their response ratio.The Response Ratio is Calculated as->(W+B)/B.(where W-current Waiting Time, B-Burst Time)The process starvation is low and no convey effect.
